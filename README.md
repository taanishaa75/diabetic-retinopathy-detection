# Diabetic Retinopathy Detection

This project detects diabetic retinopathy from retinal images using deep learning and provides an interactive dashboard.

## Features
- Retinal image classification
- CNN model (MobileNetV2)
- Heatmap visualization
- Streamlit dashboard

## Technologies
- Python
- TensorFlow
- Streamlit
- OpenCV

## How to Run

pip install -r requirements.txt
streamlit run dashboard.py
